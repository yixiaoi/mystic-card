
# Mystic Card｜神秘学扩列图生成器 PRO

一个纯前端（HTML + Canvas）的免费小工具：填写基本信息、选择标签与体系，自动生成一张“扩列卡片”，可一键导出高清 PNG。

👉 在线使用： https://yixiaoi.github.io/mystic-card/

---

## 功能 Features

- **一键生成扩列图**：头像、昵称、性别/取向、个性签名、联系方式、标签体系
- **丰富标签库**：东方 / 西方 / 新纪元三大体系，多层级分类，支持自定义标签
- **主题色与明暗模式**：可切换 Dark / Light，主题色实时预览
- **可选个人展示图**：上传一张照片作为卡片底部展示图
- **高清导出**：
  - PC：一键下载（默认 2x）
  - 移动端：保存预览弹窗（1x / 2x / 4x），支持长按保存

---

## 使用说明 How to Use

1. 打开网页： https://yixiaoi.github.io/mystic-card/
2. 左侧填写基础信息（头像/昵称/联系方式/签名）
3. 右侧选择标签体系与类别（支持展开/折叠与自定义）
4. 点击 **保存卡片** 导出 PNG  
   - 手机端若按钮无反应，可按提示 **长按预览图保存**

## 本地运行 Local Development

本项目为纯静态页面，无需构建工具。

方式 1：直接双击打开  
- 下载/克隆后直接打开 `index.html`

方式 2：起一个本地静态服务器 
```bash
# Python 3
python3 -m http.server 5173
# 然后访问 http://localhost:5173
````
